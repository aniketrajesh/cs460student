RO.ME "3 DREAMS OF BLACK"

RO.ME is a interactive movie which allows the user to play around with the environment through the use of the cursor. Wherever the the user points with the cursor on the screens, the animation of the environment changes. For example, a green animation is created and animals follow the path of the green animation as the user navigates the cursor along the screen. An interactive movie like this helps the user to be engaged with whats happening in the movie.

Technology: video, 2D graphics and 3D graphics are all playing together with the help of WebGL. WebGL helps transform your browser (google chrome) into an open source video game console. What we are seeing while interacting with the video is real time rendering of visual models. HTML and JavaScript are the two essential languages used.
